# Activity Collection

A collection of interactive web activities with modern styling and responsive design.

## Features

- Multiple activity pages with consistent theming
- Modern pink and dark theme design
- Responsive layout for all screen sizes
- Login-based access for Activity 7

## Getting Started

Open `index.html` or `act 7.html` directly in your browser to get started.

## Files

- `index.html` - Main entry point (redirects to Activity 7)
- `act 7.html` - Primary activity page
- `act 2.html`, `act 3.html`, `act 4.html`, `act 5.html` - Additional activities
- `theme.css` - Shared theme and styling

## Theme

- **Primary Colors**: Pink (#ff4da6), Black (#0a0a0a), White (#f5f5f5)
- **Typography**: Georgia serif font
- **Design**: Modern grid-based layout with dark mode aesthetic
